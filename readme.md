# Vagrant
環境はVagrantで構築。  
`Vagrantfile`内にて、IPとホスト名を好きなものに変更して`$ vagrant up`するだけ。

`https`対応済だが、チュートリアル途中のAPIへajaxでアクセスする箇所は`http`でアクセスしないとエラーになるので注意。
